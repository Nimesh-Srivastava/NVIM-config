# NVIM Config

## Windows

### For native
Install nvim and place these contents in ```C: / Users / <Username> / Local / Appdata / nvim```

### For WSL
1) Open wsl 
```wsl```
2) Go to home dir 
```cd ~```
3) Go to *.config/*, if it does not exist, create it 
```mkdir .config 
    cd .config``` 
4) Create nvim dir inside .config 
```mkdir nvim```
